## English
This repository is a fork of the original [guacamole-docker-compose](https://github.com/boschkundendienst/guacamole-docker-compose) project. The primary goal of this fork is to adapt the provided scripts and configurations, originally designed for Linux environments, to work seamlessly on Windows systems.

These adaptations include adjustments to file paths, permissions, and script execution to ensure compatibility with Windows, including support for Docker Desktop running with WSL2. This allows users on Windows to deploy and manage an Apache Guacamole instance with ease.

Feel free to explore the changes and contribute to further improvements!

## Português

Este repositório é um fork do projeto original [guacamole-docker-compose](https://github.com/boschkundendienst/guacamole-docker-compose). O objetivo principal deste fork é adaptar os scripts e configurações fornecidos, originalmente projetados para ambientes Linux, para funcionarem perfeitamente em sistemas Windows.

Essas adaptações incluem ajustes em caminhos de arquivos, permissões e execução de scripts para garantir a compatibilidade com o Windows, incluindo suporte ao Docker Desktop executando com WSL2. Isso permite que usuários no Windows implantem e gerenciem uma instância do Apache Guacamole com facilidade.

Sinta-se à vontade para explorar as mudanças e contribuir com melhorias adicionais!
